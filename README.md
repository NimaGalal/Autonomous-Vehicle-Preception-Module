# Dataset
• Name: GTSRB : German Traffic Sign Recognition Benchmark
• Training images: 39,209
• Test images: 12,630
• Classes: 43

## Phase 1 Results
- Naive Bayes Accuracy: 19%
- KNN (k=5) Accuracy: 55.8%

# Phase 2 Results
• Deep CNN Accuracy: 98.3%
• Best Optimizer: Adam (98.80%)
• Transfer Learning Fine-tuned Accuracy: 96.18%
• AutoEncoder: Stable MSE convergence over 20 epochs
